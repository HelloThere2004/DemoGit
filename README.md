## 🛠️ Quick Start (Try it out!)

Open your terminal and run these commands to get started:

```bash
# 1. Clone this repository to your local machine
git clone [https://github.com/HelloThere2004/DemoGit.git](https://github.com/HelloThere2004/DemoGit.git)
cd DemoGit

# 2. NEVER code on main! Create your own branch
git checkout -b feature/your-name

# 3. Make some changes in the files, then stage and commit them
git add .
git commit -m "My first commit at ITea LAB"

# 4. Push your branch to GitHub
git push -u origin feature/your-name
