What You'll Learn

This hands-on workshop covers the essentials of Git and GitHub:

  • git init — Initialize a new repository
  • git add & git commit — Stage and save changes
  • git branch & git checkout — Work with branches
  • git merge & git rebase — Combine work from different branches
  • git push & git pull — Sync with remote repositories
  • Pull Requests — Collaborate via GitHub

Workshop Structure

Each HTML page in this project was created in a separate branch. You'll practice:

  • Creating feature branches for each page
  • Making commits with meaningful messages
  • Merging branches back into main
  • Resolving merge conflicts (if any!)
